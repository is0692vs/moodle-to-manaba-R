# インストール方法

Moodle to manaba Timetable 拡張機能をインストールする方法を説明します。

## 必要なもの

- Google Chrome または Chromium ベースのブラウザ（Edge、Brave 等）

## インストール手順

### 1. 拡張機能ファイルのダウンロード

1. [Releases](https://github.com/is0692vs/moodle-to-manaba-R/releases)ページにアクセス
2. 最新バージョンの`moodle-to-manaba-extension-vX.X.X.zip`をダウンロード
3. ダウンロードした zip ファイルを展開（解凍）

### 2. Chrome に拡張機能を読み込む

1. Chrome を開き、アドレスバーに `chrome://extensions/` と入力してアクセス
2. 右上の「デベロッパーモード」を ON にする
3. 「パッケージ化されていない拡張機能を読み込む」ボタンをクリック
4. 展開したフォルダを選択

### 3. 使い方

1. あなたの大学の Moodle マイページ（例: `https://moodle.your-university.edu/my/`）にアクセス
2. 自動的にコースリストが manaba スタイルの時間割表に変換されます

## 注意事項

- 初回アクセス時、コース情報の取得に少し時間がかかる場合があります
- 時間割情報が設定されていないコースは表示されません
- 拡張機能を更新する場合は、古いバージョンを削除してから新しいバージョンをインストールしてください

## トラブルシューティング

### 時間割が表示されない

- ブラウザをリロード（F5 キー）してみてください
- `chrome://extensions/`で拡張機能が有効になっているか確認してください
- Moodle のマイページ（コース一覧ページ）にいることを確認してください

### 拡張機能が読み込めない

- zip ファイルを正しく展開できているか確認してください
- 展開したフォルダ内に`manifest.json`ファイルがあることを確認してください

### その他の問題

問題が解決しない場合は、[Issues](https://github.com/is0692vs/moodle-to-manaba-R/issues)で報告してください。
